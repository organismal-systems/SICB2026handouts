Activity 2: Data collection

The following steps take you through Activity 1.

Parameters¶

Set: nsizes=16n_{sizes}=16nsizes​=16, zm=100z_m= 100zm​=100, npers=128n_{pers}=128npers​=128.

1. Winners of competition in diverse diatom cultures¶

In a spreadsheet, start a table in which to record the winners of competitions in mixed layers with different mixing intervals:

Data are collected in a nested loop:

  • For each mixing interval, Tmix=1.5,8,18,32T_{mix} = 1.5, 8, 18, 32Tmix​=1.5,8,18,32:

    • Execute the simulation sequence:
      Start a population →\rightarrow→ seed “all” size classes →\rightarrow→ run the simulation →\rightarrow→ record the winner Important:

  • Make sure to run the simulation enough times to arrive at the true dominant variant – this can take multiple runs.

  • Record the winner by entering its size class under “i_win” and its size under “s_win”.

  • Make sure to record the approximate number of days the model took for a dominant strain to emerge.

Part 2: Do “winners” of competitions also form ESS’s?¶

Do a new series of runs, to fill out the table column labeled “ESS”.

Data are collected in a nested loop:

  • For each mixing interval, Tmix=1.5,8,18,32T_{mix} = 1.5, 8, 18, 32Tmix​=1.5,8,18,32:

    • Execute the simulation sequence, in two phases:

      1. Start a population →\rightarrow→ seed the dominant size class, IwinI_{win}Iwin​ →\rightarrow→ run the simulation

      2. Seed “all” size classes →\rightarrow→ run the simulation →\rightarrow→ assess if any variants grow

      In the first phase, the candidate ESS variant establishes the nutrient environment. In the second phase, other variants are tested to see whether they can grow in this nutrient environment.

  • Assess whether the “i_win” variant is an ESS, with a “y” or “n” in the ESS column.

    • If any variant is growing (that is, if it has more cells than you seeded) then “i_win” is not an ESS.

    • If no variant is growing then “i_win” is an ESS.
