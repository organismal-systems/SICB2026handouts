Understanding model output

The DiatomSizeESS model outputs two tables of results, and two sets of plots, after each simulation run.

Tabular results¶

Tabular results are shown in the noteboook cell under the Run simulation button, and also saved in the text files standingstocks.txt and losses.txt[1].

In these tables, the columns represent: iii - size class, sss - size, 10s10^s10s - cell volume, NNN - number of cells, Q/QminQ/Q_{min}Q/Qmin​ - fractional nutrient quota, BVBVBV - biovolume, MMM - “background” mortality, SSS - sinking mortality, LLL - mixing event mmortality

Graphical results¶

Two sets of graphical results are presented, each of which shows:
- Top: fractional quotas (Q/QminQ/Q_{min}Q/Qmin​)
- Middle: biovolume (BVBVBV)
- Bottom: mixed layer nutrient concentration (RRR).

  1. Period-averaged results, for all mixing intervals in the simulation.
    These plots show the long-term trends of diatom populations and nutrient. “Period-averaged” means the variation between mixing events is averaged out, so that the long-term trends are easier to see.

  2. Results during the last mixing interval.
    Variations during the last mixing interval. These plots show the transient response to a mixing event, which removes a fraction of the population but also adds nutrient to support new growth.

Footnotes¶

  1. Note that these files are over-written each time the simulation is run.
